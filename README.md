# محفظة أعمال استخبارات التهديدات | Threat Intelligence Portfolio

هذا المستودع يعرض مشاريع عملية في تحليل استخبارات التهديدات السيبرانية، إثراء مؤشرات الاختراق IOCs، وربط أساليب المهاجمين بإطار MITRE ATT&CK.

## نظرة عامة

تطبيق عملي لدورة حياة استخبارات التهديدات CTI Lifecycle: جمع المؤشرات، تحليل حملات البرمجيات الخبيثة، تتبع البنية التحتية للمهاجمين، وإنتاج تقارير قابلة للتنفيذ لفرق مركز العمليات الأمنية SOC.

### 1. حملة case study Nobelium - الربع الأول 2024

**الهدف:** تتبع وتوثيق حملة تصيد موجه تستهدف جهات حكومية.

**الربط مع case study:**
- **T1566.001**: التصيد بالمرفقات - ملف DOCX خبيث يحتوي ماكرو
- **T1059.001**: PowerShell - تحميل حمولة مشفرة
- **T1071.001**: بروتوكولات الويب - قناة C2 عبر HTTPS

**النتيجة القابلة للتنفيذ:** تم إنشاء قاعدة YARA + قاعدة KQL وتسليمها لفريق الـ SOC. [تقرير مفصل عند الطلب](./reports/APT29-Nobelium-Report.pdf)

### 2. إثراء مؤشرات برمجية الفدية BlackCat

**الهدف:** إثراء المؤشرات المسربة وتوفير سياق للحظر.

**الخطوات:**
1. جمع 50+ بصمة ملف من موقع Ransomware.live
2. إثراء آلي عبر VirusTotal API + AbuseIPDB
3. وسم الأساليب TTPs وإنشاء ملف CSV للإدخال في الجدار الناري و EDR
